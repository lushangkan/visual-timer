# Visual Timer

**Visual Timer** 是一个基于 **Tauri** + **React** 构建的跨平台可视化倒计时工具。它模拟了经典的物理视觉计时器，通过醒目的红色扇形面积直观地展示剩余时间，帮助用户缓解“时间盲区（Time Blindness）”，提升专注力。

## 🎯 项目目标 (Project Goal)

将直观的物理计时器体验带到桌面端（Windows/macOS/Linux）和移动端（Android/iOS），提供一致、极简且无干扰的时间管理体验。

## ✨ 核心功能 (Key Features)

### 1. 核心倒计时
* **可视化表盘**：最大支持 **60 分钟**。随着时间流逝，红色扇形区域会顺时针减少，剩余时间一目了然。
* **直观交互**：
    * **滑动设定**：用户可以直接在表盘上拖拽或滑动指针来快速设置倒计时时长。
    * **即时反馈**：拖拽结束后自动就绪。
* **基础控制**：
    * **启动/暂停**：随时控制计时的进行与停止。
    * **快速加时**：提供 `+1 分钟` 快捷按钮，方便临时延长专注时间。
    * **结束重置**：倒计时结束后一键归零。

### 2. 提醒机制
* **结束闹铃**：倒计时结束时播放提示音。
* **视觉提示**：倒计时归零时，红色区域完全消失并伴随界面视觉反馈。

## 🖥️ 桌面端专属体验 (Desktop Experience)

为了让用户在使用电脑工作时能保持专注且不被干扰，桌面版具有以下特性：

* **始终悬浮 (Always on Top)**：窗口保持在所有应用的最顶层，确保时间进度始终可见。
* **极简窗口**：
    * **1:1 比例锁定**：窗口始终保持正方形，模拟真实的物理时钟外观。
    * **无边框/透明设计**：视觉上仅保留圆形表盘，去除多余的系统窗口边框。
    * **自由拖动**：按住表盘任意非交互区域即可移动窗口位置。
* **尺寸限制**：支持缩放，但设有最小尺寸限制，防止窗口过小导致无法操作。

## 📱 移动端体验 (Mobile Experience)

* 保持与桌面端一致的视觉风格与交互逻辑，提供触屏友好的操作体验。
